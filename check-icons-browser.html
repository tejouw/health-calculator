Tarayıcıda bu adımları izle:

  1. Günlük Yaşam sayfasını aç
  2. Sağ tıkla > "Öğeyi İncele" (Inspect)
  3. Console'u aç (F12)
  4. Şu kodu yapıştır ve çalıştır:
// Günlük Yaşam iconlarını kontrol et
const dailyLifeIcons = document.querySelectorAll('.lucide');
console.log('Toplam icon sayısı:', dailyLifeIcons.length);
dailyLifeIcons.forEach((icon, index) => {
    const classList = Array.from(icon.classList);
    console.log(`Icon ${index + 1}:`, classList.join(' '));
});
